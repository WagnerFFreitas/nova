Requisitos da IA NOVA

Verifique se seu sistema atende aos requisitos necessários para executar a IA NOVA

Visão Geral dos Requisitos

A IA NOVA é uma aplicação de IA local que requer certos recursos de hardware e software para funcionar adequadamente. Verifique abaixo se seu sistema atende aos requisitos necessários.

CPU

Mínimo: 8 núcleos
Recomendado: 12+ núcleos

RAM

Mínimo: 16 GB
Recomendado: 32 GB ou mais

Armazenamento

Mínimo: 20 GB de espaço livre
Recomendado: 100 GB de espaço livre

GPU

Opcional, mas recomendado
NVIDIA série 3000 ou superior com pelo menos 8 GB de VRAM

Sistema Operacional

Windows 10/11
macOS 11 ou superior
Linux (Ubuntu 20.04 ou superior)

Software

Python 3.8 ou superior
Visual Studio Code
Git

Requisitos de Hardware

Componente Requisito Mínimo Observações
CPU 8 núcleos Processadores Intel Core i7 ou AMD Ryzen 7 ou superior
RAM 16 GB Suficiente para modelos menores como Llama 3 8B
Armazenamento 20 GB de espaço livre SSD recomendado para melhor desempenho
GPU Opcional NVIDIA série 1000 ou superior
Rede Conexão com a Internet Necessária para atualizações e comunicação com outras IAs

Nota: Os requisitos mínimos permitem a execução da IA NOVA com modelos menores e funcionalidades básicas. Para uma experiência completa e melhor desempenho, recomendamos os requisitos recomendados.

Componente Requisito Recomendado Observações
CPU 12+ núcleos Processadores Intel Core i9, AMD Ryzen 9 ou superior
RAM 32 GB ou mais Necessário para modelos maiores e multitarefa
Armazenamento 100 GB de espaço livre SSD NVMe para melhor desempenho
GPU NVIDIA série 3000 ou superior Com pelo menos 8 GB de VRAM
Rede Conexão de banda larga Para atualizações rápidas e comunicação eficiente

Nota: Os requisitos recomendados garantem uma experiência completa e fluida com a IA NOVA, permitindo o uso de modelos maiores e todas as funcionalidades disponíveis.

Requisitos por Tamanho de Modelo

A IA NOVA é compatível com vários tamanhos de modelos de linguagem. Os requisitos de memória variam de acordo com o tamanho do modelo escolhido.

Tamanho do Modelo Memória Necessária Recomendação de Hardware
1B parâmetros ~2GB de memória Qualquer PC moderno
3B parâmetros ~6GB de memória 8GB de RAM, sem GPU
7B parâmetros ~14GB de memória 16GB de RAM ou GPU com 8GB+ VRAM
13B parâmetros ~26GB de memória 32GB de RAM ou GPU com 16GB+ VRAM
70B parâmetros ~140GB de memória GPU de alta capacidade ou múltiplas GPUs

Nota: A quantização pode reduzir significativamente os requisitos de memória, permitindo a execução de modelos maiores em hardware mais modesto, com alguma perda de qualidade.

Requisitos de Software

Sistema Operacional

  • Windows: Windows 10 ou Windows 11
  • macOS: macOS 11 (Big Sur) ou superior
  • Linux: Ubuntu 20.04 ou superior, Debian, Fedora, Arch Linux

Software Necessário

  • Python: Versão 3.8 ou superior
  • Visual Studio Code: Última versão estável
  • Git: Para controle de versão e download de dependências

Bibliotecas e Frameworks

  • Ollama: Para execução de modelos de IA local
  • MediaPipe: Para detecção e animação facial
  • PyQt5: Para interface gráfica
  • SQLite: Para armazenamento local de dados
  • Requests: Para comunicação com APIs
  • NumPy e Pandas: Para processamento de dados
  • OpenCV: Para processamento de imagem

Perguntas Frequentes sobre Requisitos

Posso executar a IA NOVA em um laptop?

Sim, a IA NOVA pode ser executada em laptops que atendam aos requisitos mínimos. Laptops com CPUs mais recentes (Intel Core i7/i9 ou AMD Ryzen 7/9) e pelo menos 16 GB de RAM são recomendados. Para modelos menores, laptops com boas especificações devem funcionar bem.

A GPU é realmente necessária?

A GPU não é estritamente necessária, mas é altamente recomendada para melhor desempenho, especialmente ao usar modelos maiores. Sem uma GPU, a IA NOVA usará a CPU para processamento, o que pode ser significativamente mais lento. Modelos menores (1-3B parâmetros) podem funcionar razoavelmente bem apenas com CPU.

Quanto espaço em disco os modelos ocupam?

O espaço em disco necessário varia de acordo com o tamanho do modelo. Modelos menores (1-3B) ocupam cerca de 2-6 GB, modelos médios (7-13B) ocupam cerca de 14-26 GB, e modelos grandes (70B) podem ocupar mais de 140 GB. A quantização pode reduzir esses tamanhos.

A IA NOVA funciona em computadores Mac com Apple Silicon (M1/M2)?

Sim, a IA NOVA é compatível com computadores Mac com Apple Silicon (M1/M2/M3). Na verdade, esses processadores têm um bom desempenho para execução de modelos de IA localmente, graças à sua Neural Engine integrada. Recomendamos pelo menos 16 GB de RAM para uma experiência ideal.

Posso usar a IA NOVA sem conexão com a Internet?

Sim, a principal vantagem da IA NOVA é que ela funciona localmente, sem necessidade de conexão constante com a Internet. No entanto, algumas funcionalidades como atualização automática de dados e comunicação com outras IAs requerem conexão com a Internet. A funcionalidade básica de processamento de linguagem funciona completamente offline.
